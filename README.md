Usando la API de Itunes:

Using itunes API
1-The 10 last released songs by Galantis
2-The 10 last released songs by one artist you know.
3-Search for 30 media types with the word “Peace” in the title and extract their artist
name and media type.
4-All the songs released by Adele in the year 2021.
5-The 10 podcasts and movies with the best rating index.

¿Cómo ejecutar el código?
Para ejecutar el código puedes guardar la carpeta en tu 
PC y abrir el archivo index.html desde tu navegador.
También copiando y pegando el código en la siguiente web https://www.cubicfactory.com/jseditor/

Funcionamiento:
La API es muy restringida, es por eso que solicito más información 
que la requerida a la API y después segmento mediante restricciones de parámetros
para dar el resultado esperado.

Si existiese más documentación sobre la API y sus usos podría minimizarse el código
aportando los parámetros de la API necesarios, pero al ser escasa, hacerlo mediante códigoes la mejor manera de
solucionarlo que he encontrado.
